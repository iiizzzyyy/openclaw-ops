OpenClaw Self-Healing Architecture

Three-layer monitoring: 5-minute watchdog, hourly observability, daily LLM evaluation

5-Minute Watchdog

  • • Process health ping
  • • Auto-update to latest GitHub release
  • • Log-based diagnosis (API errors, OOM, config)
  • • Auto-restart with appropriate fixes
  • • Telegram alerts only on failures

Hourly Observability

  • • Queries ClawLens API for sessions, errors, costs
  • • Detects: error spikes, cost anomalies, agent failures
  • • Per-agent token breakdown
  • • Cron job health monitoring
  • • Zero noise, all signal

Daily LLM Evaluation

  • • 35 test cases across 8 quality dimensions
  • • Judge model: gemma4:31b-cloud (Ollama)
  • • Current pass rate: 51% (embarrassing but honest)
  • • Catches regressions in 24h vs 3 weeks
  • • Role-aware rubric weights (Tier 1/2/3 agents)
